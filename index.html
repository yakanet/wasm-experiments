POUSSE (OP 00)
Pousse le nombre (0-255) qui arrive après sur la stack
SOMME (OP 01)
Consomme les deux nombres sur la stack, les additionne, et pousse le résultat sur la stack
ECRIT (OP 02)
Prend le nombre qui est sur la stack et écrit dans la console le résultat
